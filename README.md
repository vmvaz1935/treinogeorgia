# Protocolo de Reabilitação - Pace

Página web do protocolo de reabilitação para a paciente Georgia.

## Como ativar o GitHub Pages

1. Acesse o repositório no GitHub: https://github.com/vmvaz1935/treinogeorgia
2. Vá em **Settings** (Configurações)
3. No menu lateral, clique em **Pages**
4. Em **Source**, selecione **GitHub Actions**
5. Salve as alterações

A página estará disponível em: `https://vmvaz1935.github.io/treinogeorgia/`

O workflow do GitHub Actions já está configurado e fará o deploy automaticamente a cada push na branch `main`.
